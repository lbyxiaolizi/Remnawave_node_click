Remnawave Node 一键安装命令生成器（从 docker-compose.yml 提取 → 调通用脚本参数）

左侧粘贴 Remnawave 生成的 docker-compose.yml，解析 services → 目标服务（默认 remnanode）→ environment， 生成推荐的一键命令： curl 脚本 | sudo bash -s -- 参数
安全提醒：不要把 SECRET_KEY 放到 URL Query 里（会进历史/代理/日志）。本页面把 SECRET_KEY 放在命令参数里，并用单引号安全转义。
等待粘贴 YAML…
从 YAML 提取到的关键信息：
NODE_PORT
-
SECRET_KEY
-
image
-
network_mode
-
nofile soft/hard
-
支持 environment 为列表（- KEY=VALUE）或字典（KEY: VALUE）。
脚本来源固定为：打开通用脚本
提示：如果你看到 “bash: <!DOCTYPE html>” 这种错误，说明你把网页链接当脚本跑了；一键命令里必须是 install-remnanode.sh 的 raw 链接。
